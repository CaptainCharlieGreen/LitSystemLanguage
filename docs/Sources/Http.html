HTTP

mostly todo

The Http source sources http events into your system.

Http's event config must have a method and a path. The method must be:

  • get
  • post
  • put
  • delete

Path examples:

  • "/"
  • "/*/foo"
  • "foo.html"
  • "/foo/:argument1/bar"
  • "/:arg1/:arg2/bar"
